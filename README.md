# Restaurant Website

A responsive restaurant website built using **HTML, CSS, and JavaScript**.

## Features

* Responsive design
* Interactive menu filtering
* Contact form
* Google Maps integration
* WhatsApp ordering button
* Click-to-call phone integration

## Pages

* Home Page
* Menu Page
* Contact Page

## Technologies Used

* HTML5
* CSS3
* JavaScript
